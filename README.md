# React Todo List

Basic Todo App built with React to practice developing with:

* Functional Components
* State Hooks
* Event Listeners
* Spread syntax for dynamic data

<img src="Assets/React-Todo-Image.png" width="600">

## Quick Start
### Install Dependencies

```
npm install
```
### Run
```
npm start
```
